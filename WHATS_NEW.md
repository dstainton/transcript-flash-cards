## üîß How to Use New Features

### Creating a Project with Lesson Files

1. Go to **Projects ‚Üí Create Project from Documents**
2. Drag all your lesson files (e.g., 32 .txt files)
3. Click "Process Documents"
4. On config screen:
   - Select **"One Topic Per File"** (default)
   - Review the 32 topics listed (one per file)
   - Adjust flashcard counts if needed (default: 25 per topic)
   - Edit topic names if desired
   - Review project name and customize
5. Click **"Create Project & Generate Flashcards"**
6. Wait ~30-40 minutes for 800 flashcards to generate
7. Start studying!

### Viewing Answer Explanations

1. Start a study session
2. Answer any question
3. Scroll down to see your completed answers
4. Look for **üí° Explanation:** section below each answer
5. Read the explanation to understand the concept

**Note:** Only NEW flashcards (created after this update) include explanations. Existing cards will show "No explanation available."

### Checking Session History

1. Answer some flashcards (any number)
2. Navigate to **Stats** or **Home** (don't need to finish or exit)
3. See flash message confirming save
4. Check Stats page - your session is recorded!
