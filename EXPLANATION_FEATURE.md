# Answer Explanations Feature - User Guide

## Overview

Every flashcard now includes **AI-generated explanations** that help you understand not just WHAT the correct answer is, but **WHY** it's correct. This feature enhances learning by providing context and reinforcement for both correct and incorrect answers.

## How It Works

### üìù AI-Generated Explanations

When new flashcards are created from your documents:

- The AI generates 1-2 sentence explanations for each question
- Explanations clarify the reasoning behind the correct answer
- They provide educational context from your source material
- All question types (True/False, Yes/No, Multiple Choice, Multiple Answer) include explanations

### üí° Where You See Explanations

1. You answer a question (correctly or incorrectly)
2. The system shows:
   - ‚úÖ/‚ùå Whether you were correct
   - Your answer vs. the correct answer
   - Feedback about your streak (in Study Mode)
   - **üí° Explanation:** The detailed explanation

### üéØ Benefits

**For Correct Answers:**
- Reinforces your understanding of the concept
- Helps solidify knowledge even when you guessed correctly
- Provides context you might not have considered

**For Incorrect Answers:**
- Explains WHY your answer was wrong
- Clarifies the correct concept
- Reduces frustration by providing learning context
- Helps you avoid the same mistake next time

## Examples

### True/False Question
- **Question:** "Python is a high-level, interpreted programming language known for its simplicity and readability."
- **Answer:** True
- **üí° Explanation:** "Python is indeed a high-level language because it abstracts away low-level details like memory management, and it's interpreted rather than compiled, making it easier to write and test code quickly."

### Multiple Choice Question
- **Question:** "Which of the following is NOT a data type supported by Python?"
- **Answer:** D) Character
- **üí° Explanation:** "Python does not have a 'Character' data type. Instead, it uses strings to represent character data. Even a single character is treated as a string of length 1 in Python."

### Multiple Answer Question
- **Question:** "Select all the key concepts of Object-Oriented Programming in Python."
- **Answer:** A,B,C (Encapsulation, Inheritance, Polymorphism)
- **üí° Explanation:** "The core concepts of Object-Oriented Programming in Python are Encapsulation (bundling data and methods), Inheritance (creating classes based on existing ones), and Polymorphism (objects of different classes being treated through a common interface). Compilation is not an OOP concept."

## Best Practices

1. **Read Every Explanation** - Even when you get the answer right
2. **Learn from Mistakes** - Use explanations to understand incorrect answers
3. **Reinforce Knowledge** - Explanations help cement correct answers in your memory
4. **Context Matters** - The explanations tie back to your source material

## Study Mode vs. Exam Mode

- **Study Mode:** ‚úÖ Explanations shown after each answer
- **Exam Mode:** ‚úÖ Explanations shown in the results review after completing the exam

This ensures you get the learning benefits in both modes while maintaining the exam simulation experience.

## Technical Details

- Explanations are generated by GPT-4o during flashcard creation
- They are stored in the `explanation` field of each flashcard
- Styled with a light blue background (üí° icon) for easy identification
- Displayed for all answer types and all modes
